🔍 Data Extraction Strategy

Multi-Selector Fallback System

Setiap field memiliki 5-20 selector alternatif untuk mengantisipasi perubahan struktur HTML.

// Example: Price extraction with multiple fallbacks
const price = findText([
  '[data-sqe="price"]', // Primary (Shopee internal)
  '.product-briefing__price--normal', // Secondary
  'meta[property="og:price:amount"]', // Meta tag fallback
  'div[class*="price"] span', // Partial match
  '._3n5NQx' // Obfuscated class
]);

🎯 Extraction Flow

1. Try primary selector (data-sqe attribute)
2. If failed → Try secondary selector (class-based)
3. If failed → Try meta tags (og:xxx)
4. If failed → Try wildcard selectors (*=partial)
5. If all failed → Return null / "NOT_FOUND"

💻 Code Walkthrough

1. Browser Launch & Configuration

const { chromium } = require('playwright');

const browser = await chromium.launch({
  headless: false, // Visible browser for debugging
  slowMo: 300, // Anti-detection delay
  args: ['--disable-blink-features=AutomationControlled']
});

const page = await browser.newPage();
await page.goto(url, { waitUntil: 'networkidle' });

2. Data Extraction inside Browser

const productData = await page.evaluate(() => {
  // This code runs INSIDE the browser
  return {
    name: document.querySelector('[data-sqe="name"]')?.innerText,
    price: document.querySelector('[data-sqe="price"]')?.innerText,
    images: Array.from(document.querySelectorAll('img')).map(img => img.src)
  };
});

🛡️ Anti-Detection Techniques

What triggers bot detection?

  • Headless browser fingerprint (missing WebGL, fonts, etc)
  • Automated navigation patterns (no mouse movements)
  • Missing browser-specific navigator properties
  • Too many requests in short time

Our Countermeasures

TechniqueImplementation
Realistic User-AgentMozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36
Viewport size1920x1080 (common desktop resolution)
Random delaysslowMo: 300 + random timeout between actions
Human-like scrollingautoScroll() with step-by-step scrolling
Remove automation flags--disable-blink-features=AutomationControlled
Cookie persistenceSave & restore session cookies

🚀 How to Run

Prerequisites

  • Node.js v18 or higher
  • npm or yarn package manager

Step-by-Step Execution

# 1. Clone / create project folder
mkdir shopee-scraper && cd shopee-scraper

# 2. Initialize project
npm init -y

# 3. Install Playwright
npm install playwright

# 4. Install Chromium browser
npx playwright install chromium

# 5. Create scrape-shopee.js (copy code)
# 6. Run the scraper
node scrape-shopee.js

# Optional: Run with custom URL
node scrape-shopee.js --url "https://shopee.sg/product/..."
